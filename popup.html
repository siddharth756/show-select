Running

The floating seiyaa panel is injected into every page you visit. Select text in any input or textarea to begin.

How It Works

  1. 1 Navigate to any webpage with text inputs
  2. 2 Select text inside an <input>, <textarea>, or contentEditable
  3. 3 The floating seiyaa panel shows your selection
  4. 4 Edit the replacement text and click Replace

Supported Elements

<input> <textarea> contentEditable role="textbox"

Works on ChatGPT, Google Docs*, Gmail, GitHub, and all standard form fields.

*Google Docs uses a custom rendering engine; contentEditable support is best-effort.

Features

  • 🔄 Real-time two-way text sync
  • 📋 One-click copy selected text
  • ✏️ Edit & replace inline
  • 🎯 Cursor position preserved
  • 📦 Draggable & collapsible
  • ⌨️ Toggle with Alt+S key
  • 🌐 Works across all websites
  • ⚡ React-compatible (execCommand)
  • 🔍 MutationObserver for SPAs
